# 📮Packet Tracer📮

### 🖥What is it?🖥

> `Ethernet Frame`을 통해 받은 `Packet`을 분석해 해당 프레임의 구조와 데이터를 분석

<br>

### 🔧Environment

- Language
    + JSP
    + HTML, CSS
    + JAVA

- IDE
    + Eclipse 2020

<br>

### 📖Interpretable Packet
- `Ethernet Frame`
- `IP`
- `ICMP`
- `ARP`
- `XNS IDP`
- `X.25 PLP`
- `RARP`
- `IPv6`
- `TCP`
- `EGP`
- `UDP`
- `OSPF`

<br>
