### Packet 분석 프로그램

- 개발언어
    + JSP
    + HTML, CSS
    + JAVA

- 개발환경
    + Eclipse 2020

<hr/>
<br><br>

- 내용
    + Packet을 분석해 사용자에게 Packet안에 들어있는 데이터를 제공

- 해석 가능 패킷
    + `Ethernet Frame`
    + `IP`
    + `ICMP`
    + `ARP`
    + `XNS IDP`
    + `X.25 PLP`
    + `RARP`
    + `IPv6`
    + `TCP`
    + `EGP`
    + `UDP`
    + `OSPF`
